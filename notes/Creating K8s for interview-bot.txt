Creating K8s for interview-bot

Install minikube: 

Start minikube with the Driver Docker: 
minikube start --driver=docker

Enable Minikube Docker Environment:
This allows you to use the Docker daemon inside Minikube.
eval $(minikube docker-env)

Buildx not suported by docker driver so normal image builds: 
docker build -t interview-app:minikube .
docker pull postgres:13

kubectl apply -f postgres-pvc.yaml
kubectl apply -f postgres-deployment.yaml
kubectl apply -f postgres-service.yaml

kubectl apply -f uploads-pvc.yaml
kubectl apply -f app-deployment.yaml
kubectl apply -f app-service.yaml

http://192.168.49.2:30001

kubectl port-forward service/interview-app 5000:5000
